# Debt Downshift

Debt Downshift is a lightweight, single-page debt payoff planner. It lets you enter income, essential expenses, recurring bills, and multiple debts, then compares avalanche vs snowball strategies to estimate payoff time, total interest, and a month-by-month plan.

The app also supports one-time payments, CSV import/export, printable summaries, and a simple progress dashboard with streaks and milestones to keep the plan up to date.

## Project Structure

- `index.html` — the full UI and layout
- `styles.css` — visual styling
- `script.js` — calculations, charts, and interactions
